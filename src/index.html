Systembolaget+

It's hard to know what's good when browsing Systembolaget. This Chrome extension adds Vivino ratings to wine, Untappd ratings to beer and Distiller ratings to liquors. Helping me make better decisions.

facepause

Look away to pause YouTube videos. When Chrome released its FaceDetector API, I built this experiemntal extension to see how good it was. It turns out it works pretty well. Removed from the Chrome web store, installing manually might still work.

Springsteen

Spring physics makes animations on the web more fluid and organic. But it's not easy to understand the outcome based on the input values while coding. This interface let’s you find the perfect combination of values for anime.js.
